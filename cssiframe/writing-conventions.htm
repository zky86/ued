CSS书写规范

单个类写法

  
    /* 单个类写法 */
    .poco {text-align: center; width: 300px;}

多类共享代码写法

   
    /* 多类共享代码写法 */
    .poco-tag1, .poco-tag2 {margin: auto; text-align: left;}

关系写法

	
    /* 关系写法 */
    .nav {width: 500px; height: 400px;}
    .mainnav {width: 500px; height: 300px;}
    .subnav {width: 500px; height: 100px;}
