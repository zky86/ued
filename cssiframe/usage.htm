引用说明

  1. 一般情况下，每个项目均需引入 reset.css ；
  2. POCO CSS 包含 reset.css + layout.css + typography.css + grids.css + image.css + header_v2.css(新版头部) + footer.css ；
  3. 每个CSS文件均有相应的压缩优化版本，命名采用“文件名-min.css”形式，eg：reset-min.css；
  4. 大部分情况下，请直接引用以下文件，此文件整合上述所罗列的css文件：

    http://www.poco.cn/css_common/pcsp.css [查看]

  5. 只需要 reset、layout 和 typography 时，请引用：

    http://www.poco.cn/css_common/reset-layout-typography.css [查看]

  6. 只需要 reset、layout、typography 和 grids 时，请引用：

    http://www.poco.cn/css_common/reset-layout-typography-grids.css [查看]

  7. 附上CSS文件地址：
        reset：http://www.poco.cn/css_common/reset.css  [查看]
        layout：http://www.poco.cn/css_common/layout.css  [查看]
        typography：http://www.poco.cn/css_common/typography.css  [查看]
        grids：http://www.poco.cn/css_common/grids.css  [查看]
        image：http://www.poco.cn/css_common/image.css  [查看]
        header：http://www.poco.cn/css_common/header_v2.css  [查看]
        footer：http://www.poco.cn/css_common/footer.css  [查看]
