1.页头

    以下为常用样式，未列举的请参考header.css
    
    更改左侧三个连接的样式
         .header_link {
             top: 56px;
             left: 0;
             width: 412px;
             height: 61px;
             background: url(header_link_bg.gif) left top no-repeat;
         }
        .header_link_1，.header_link_2，.header_link_3 分别对应三个连接
    更改快捷链接文字样式
        .quick_link li {
            color: #FFF;
        }
        .quick_link a {
            color: #FFF;
            text-decoration: none;
        }
    更改主菜单文字样式
        .menu li {
            color: #000;
        }
        .menu li a {
            text-decoration: none;
        }
    更改二级菜单样式
        /* 改变下拉菜单的箭头 */
        .submenu {
            background: url(../images/index_v2/submenu_triangle.gif) center top no-repeat; 
        }
        /* 改变下拉菜单背景色 */
        .submenu ul{
            background-color: #FF0099;
        }
        /* 改变下拉菜单分隔条样式 */
        .submenu li {
            border-bottom: #000 dashed 1px;
        }
        /* 改变下拉菜单链接样式 */
        .submenu li a {
            text-decoration: none;
        }
    更改主导航样式
        .header_mainnav li {
            color: #000;

2.页尾

    以下为常用样式，未列举的请参考footer.css
    
    //============================搜索区域部分=======================//
    页尾需要用到背景的都写在这里，方便修改
	#footer_search .footer_search_main,
	#footer_search .footer_btn_search,
	#footer_search .footer_link_top,
	#footer .sitemap_my,
	#footer .sitemap_help {
		background-image: url(http://www.poco.cn/module_common/footer/footer_bg.jpg);
	}   
    搜索区域背景色(一般设置为与body相同的背景色)
         #footer_search,
         #footer_search .footer_search_main {
             background-color: #FF0099;
         }
    搜索标签样式
        #footer_search .footer_search_main li { 
             color: #FFF;
        }
    当前搜索标签样式
        #footer_search .current_search_tag { 
             background-color: #FF64BB;
        }
    热门关注样式
        #footer_search .footer_hot_concern { 
             color: #FFF;
        }
        #footer_search .footer_hot_concern a {
             color: #FFF;
             text-decoration: underline;
        }
    //============================sitemap部分=======================//
    标题样式
        #footer .sitemap strong {
             color: #FF0099;
        }
    链接样式
        #footer .sitemap a {
             color: #000;
